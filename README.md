# Recipe Manager

Учебный проект по объектно-ориентированному программированию на Python.

Проект реализует систему для работы с рецептами.

## Установка

```bash
git clone https://github.com/ShadowFiend1337/OOP_HW_TP.git
cd OOP_HW_TP
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Использование

Основная реализация находится в файле `recipes.py`.

Для запуска тестов выполните:

```bash
pytest
```
